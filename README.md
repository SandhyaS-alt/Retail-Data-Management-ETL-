RETAIL DATA MANAGEMENT 
 
1. This project enhanced ABC Retail's data management capabilities using AWS   
     Glue, an ETL service, to streamline data processing workflows and  
     Derived actionable insights from sales and product data. 
2. It aims to integrate and analyse data from multiple sources to understand  
    customer behaviour, product performance, and market trends. This  
    integration leads to data-driven decisions that optimize inventory  
    management, marketing strategies, and overall business performance. 
3. Actions taken are :-
   * Logged in to the AWS Console 
   * Created two subfolders inside the etl-cep-sandhya-01 bucket 
   * Created a new bucket 
   * Navigated to AWS Glue and created a new database 
   * Set up two classifiers to read transaction data and product data 
   * Created IAM role 
   * Set up a Crawler 
   * Created ETL job 
   * Checked the output and run the sql query
